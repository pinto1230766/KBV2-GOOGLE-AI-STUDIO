# KBV2-GOOGLE-AI-STUDIO

## Description
KBV Lyon is a comprehensive event management application built with Google AI Studio. It provides tools for managing speakers (Orateurs), planning events, tracking accommodations, managing settings, and monitoring communications.

## Features
- **Dashboard**: Overview of key performance indicators and event summary
- **Planning**: Event scheduling and management
- **Orateurs (Speakers)**: Speaker database with detailed profiles and history
- **Hébergement (Accommodations)**: Room and accommodation management
- **Réglages (Settings)**: Application configuration and preferences
- **Dark Mode**: Theme customization
- **WhatsApp Integration**: Direct messaging capabilities
- **Message Management**: Communication tracking
- **Conflict Resolution**: Event scheduling conflict detection

## Technologies
- Google AI Studio (Frontend)
- Google Sheets (Database)
- TypeScript/React

## Getting Started
1. Open the application in Google AI Studio
2. Navigate through the tabs to access different features
3. Manage events, speakers, and accommodations

## Project Structure
The application is organized into the following main sections:
- Dashboard: KPI tracking and statistics
- Planning: Event scheduling
- Orateurs: Speaker management
- Hébergement: Room allocation
- Réglages: System settings
